# CarCrush
A simple game using SDL2

<p align="center">
  <img alt="CarCrush" src=".github/carcrush.png">
</p>

## Build
You need `SDL2` and `SDL2-Image` instaled, then:

    make

## Run 
Simple run:

    make run
